# Hand Gesture Recognition

A computer vision project built using Python, OpenCV, and MediaPipe for real-time hand tracking and gesture recognition.

## Features

- Real-time hand tracking
- Finger counting
- Left/Right hand detection
- Multi-hand support
- Hand landmark visualization

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run

pip install -r requirements.txt

python hand_tracking.py

python gesture_recognition.py
